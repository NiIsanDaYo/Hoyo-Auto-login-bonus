# HoYoLab ログインボーナス 自動取得

HoYoLabのログインボーナスをワンクリックで自動取得するChrome拡張機能です。

## 対応ゲーム

| ゲーム             | 英語名            |
| ------------------ | ----------------- |
| 原神               | Genshin Impact    |
| 崩壊：スターレイル | Honkai: Star Rail |
| ゼンレスゾーンゼロ | Zenless Zone Zero |

## 機能

- **ワンクリック取得** — ポップアップからゲームを選んでクリックするだけで、ログインボーナスを自動取得
- **一括取得** — 「すべて取得」ボタンで全ゲームのボーナスをまとめて取得
- **取得済み判定** — すでに取得済みの場合は通知で知らせてくれます
- **自動タブ閉じ** — 取得完了後、タブを自動で閉じます
- **通知バナー** — 取得結果をページ上部にバナーで表示

## インストール方法

1. このリポジトリをダウンロードまたは `git clone` する
2. Chromeで `chrome://extensions` を開く
3. 右上の **「デベロッパーモード」** を有効にする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. ダウンロードしたフォルダを選択する

## 使い方

1. **事前準備**: ブラウザで [HoYoLab](https://www.hoyolab.com/) にログインしておく
2. ツールバーの拡張機能アイコンをクリック
3. 取得したいゲームのカードをクリック、または **「すべて取得」** をクリック
4. バックグラウンドでタブが開き、自動でボーナスが取得されます
5. 取得完了後、タブは自動で閉じます

## 技術詳細

- **Manifest V3** 対応
- **権限**: `tabs`（タブの作成・閉じに使用）
- **Content Script**: `act.hoyolab.com` 上で動作し、ボーナスの自動クリックを実行
- **Background Service Worker**: Content Scriptからの完了報告を受け取り、タブを閉じる

### 自動取得の仕組み

1. ポップアップからゲームのログインボーナスページをバックグラウンドタブで開く
2. Content Scriptがページの読み込み完了を待機
3. 未取得のアイテムを検出（背景画像の差異・テキスト色・CSSクラスなど複数の方法で判定）
4. アイテムをクリックして取得
5. 結果を通知バナーで表示し、Background Service Workerにタブ閉じを依頼

## ⚠️ 注意事項

- HoYoLabに**事前にログイン**している必要があります
- HoYoLabのサイト構造が変更された場合、動作しなくなる可能性があります
- 本拡張機能はHoYoverse公式ツールではありません
